# Chat UI "OpenAI Theme" 设计文档

## 1. 设计目标
- 借鉴 ChatGPT 的清爽、极简风格：浅灰背景，白色对话框，绿色强调色。
- 减少饱和度高的紫色，使用更加中性、专业的配色。
- 保持 Tailwind + Ant Design 技术栈，尽可能通过 `@apply` 与主题变量实现。

## 2. 核心调整
| 区域            | 调整                                                        | 说明                     |
| --------------- | ----------------------------------------------------------- | ------------------------ |
| 主色 `primary`  | 从紫色 `#7c3aed` → 绿色 `#0ea472`                           | 与 OpenAI 品牌色保持一致 |
| 背景            | `body` 背景 `#f8fafc` 保持；聊天气泡内外留白加大            | 模仿 ChatGPT 大间距设计  |
| User 气泡       | 绿色背景 + 白字                                             | 区分角色同时不过于刺眼   |
| Assistant 气泡  | 白底 + 灰边                                                 | 保持阅读的舒适性         |
| Tool Call Block | 统一灰底 + 细边框；成功/失败/加载用浅绿色/红色/蓝色边线区分 | 信息层级更加清晰         |
| 折叠指示        | `<summary>` 右侧箭头旋转动画                                | 符合直觉                 |

## 3. Tailwind 配置
在 `tailwind.config.js` 的 `extend.colors.primary` 内替换为绿色系 10% 阶梯。（已实现）

## 4. 全局 CSS 变量
添加 `:root --color-primary-xxx` 以便未来非 Tailwind 环境也可使用。（已实现）

## 5. 组件覆盖
- 更新 `.chat-bubble-user` 使用 `bg-primary-600`。（已实现）
- 为 `.tool-call-block summary` 去掉默认箭头，改为自定义 `▸`，并在 `[open]` 时旋转。（已实现）

## 6. 动效
使用 `transition: transform 0.2s` 处理箭头旋转，`transition: all 0.2s` 让折叠动画更平滑。

## 7. 验证 Checklist
1. 打开聊天页，整体配色与 ChatGPT 近似；
2. 发送消息，用户气泡绿色，助手气泡白色；
3. 有工具调用输出时默认折叠，点击展开，箭头旋转；
4. 黑暗模式未来🌓（暂未实现）。
