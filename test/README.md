# 聊天功能测试文档

本目录包含了完整的聊天功能测试套件，用于验证聊天页面的所有功能是否正常工作。

## 测试文件说明

### 1. chat_functionality_test.html
**交互式Web测试页面**

这是一个完整的HTML测试页面，提供了图形化界面来测试所有聊天功能。

**功能特性：**
- 🔐 用户登录测试
- 🤖 模型列表获取和选择
- 📝 聊天历史管理（获取、查看、删除）
- 💬 实时流式聊天测试
- ⚠️ 错误处理验证
- 🔄 综合集成测试

**使用方法：**
1. 启动开发服务器：`npm run dev`
2. 在浏览器中打开：`http://localhost:5173/test/chat_functionality_test.html`
3. 按照页面提示逐步进行测试

### 2. run_chat_tests.js
**自动化测试脚本**

Node.js 自动化测试脚本，可以快速验证后端API的基本功能。

**使用方法：**
```bash
# 安装依赖（如果还没有安装）
npm install axios form-data

# 运行测试
node test/run_chat_tests.js
```

## 故障排除

### 常见问题

**1. 登录失败**
- 检查用户凭据是否正确
- 确认后端用户管理服务是否运行
- 验证config.yaml中的user_manage_url配置

**2. 模型列表为空**
- 检查模型聊天服务是否运行
- 验证config.yaml中的model_chat_url配置
- 确认后端模型服务正常

**3. 流式聊天无响应**
- 检查选择的模型是否有效
- 确认网络连接稳定
- 查看浏览器开发者工具的网络面板

**4. 聊天历史加载失败**
- 确认用户ID和会话ID正确
- 检查后端数据库连接
- 验证历史记录是否存在

### 调试技巧

**1. 启用详细日志**
```javascript
// 在浏览器控制台中启用调试
localStorage.setItem('debug', 'true');
```

**2. 检查网络请求**
- 打开浏览器开发者工具
- 切换到Network面板
- 查看API请求和响应详情

**3. 后端日志检查**
```bash

npm run dev
```
