🚀 AI 聊天界面 - 简化版使用说明

📋 功能说明：
- 初始化：加载配置文件，读取 API URL
- 发请求：获取模型列表，更新全局变量
- 模型选择：点击下拉菜单选择 AI 模型

🔧 使用步骤：

1. 安装 Node.js
   - 访问 https://nodejs.org/
   - 下载 LTS 版本并安装

2. 安装依赖
   - 双击 "安装依赖.bat"

3. 启动服务器
   - 双击 "启动服务器-nodejs.bat"
   - 浏览器自动打开 http://localhost:3000

📁 文件说明：
- server.js        # 简化的 Node.js 服务器
- scripts.js       # 简化的前端代码
- config.yaml      # 你的 API 配置（不需要修改）
- package.json     # 项目依赖

🔄 工作流程：
1. 页面加载 → 调用 init() 函数
2. init() → 加载配置文件 → 更新全局变量
3. 点击模型按钮 → 调用 loadModels() → 获取模型列表
4. 选择模型 → 更新全局变量 selectedModelId

💡 新手友好：
- 代码简化，只保留核心功能
- 注释清晰，容易理解
- 适合本地和局域网使用
- 自动处理错误，有模拟数据兜底

❓ 遇到问题：
- 检查 Node.js 是否安装
- 确保 3000 端口没被占用
- 查看控制台输出的错误信息
